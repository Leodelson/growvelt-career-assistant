## AI Development Workflow

Growvelt Career Assistant was developed during OpenAI Build Week using OpenAI tools together with manual implementation in Visual Studio Code.

### GPT-5.6

GPT-5.6 served as the primary AI development assistant throughout the project. It was used to:

- Brainstorm product features and user workflows.
- Design the application architecture.
- Generate and improve React, Next.js, and TypeScript code.
- Build responsive UI components with Tailwind CSS.
- Debug implementation issues and resolve errors.
- Improve accessibility, responsiveness, and overall user experience.
- Assist with documentation, project content, and deployment guidance.

### Codex

Codex was used during development to inspect portions of the codebase, review implementation approaches, and help refine selected components. Due to usage limits, most coding assistance came from GPT-5.6, while Codex was used selectively for code review and improvement.

### Development Process

All project decisions, feature integration, testing, debugging, and final implementation were completed by the project author. AI tools accelerated development by providing code suggestions, technical guidance, debugging assistance, and architectural recommendations, while all final code was reviewed, adapted, and integrated manually.
